STEPS to Follow:
1. Copy web-sample.config to web.config
2. Edit web.config according to your setup.
3. For Aadhaar OTP POST to YourURL/AadharOTP.aspx
4. For Aadhaar KYC Authentication POST to YourURL/AadhaarAuthentication.aspx
5. For Aadhaar KYC Data POST to YourURL/AadhaarKYC.aspx
6. For Aadhaar Demographic Authentication POST to YourURL/DemographicAuth.aspx
7. Give proper permission to the log files.
